## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Ruby should be install in your machine.

### Setup

```
git@github.com:ArnaudBand/ruby_capstone.git
```

```
cd ruby_capstone
```

### Install

```
 bundle install
```

### Run Linters

```
rubocop
```

### run test

```
rspec spec
```
